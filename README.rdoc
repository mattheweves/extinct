== EXTINCTION FACTS SAMPLE APP

This is a sample Rails App with a simple database structure, holding quotes about Extinction and randomly displaying them on the home page. 

Purpose is to test/learn:

* MVC Framework

* Sample Controller

* Sample Routes

* Database creation/migration

* Database initialization

* Accessing Data using Ruby and randomly selecting items in the db

* CSS/Web Page Layout

* Deployment instructions

* Deployment to Heroku
